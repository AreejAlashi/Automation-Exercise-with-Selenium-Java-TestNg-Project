# Automation Exercise with Selenium, Java, and TestNG

This is an automated testing project using:
- Java
- Selenium
- TestNG
- Full manual documentation

The automation was performed on the "Automation Exercise" website using Eclipse IDE for product part using POM design.

All Test Cases, Bug Reports, Summary Report, Test Plan, and automation code were created by me, as my personal contribution to the project, covering the Product module and part of the Cart module.
